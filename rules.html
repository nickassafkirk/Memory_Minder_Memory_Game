Instructions

Image carousel

Memory minder is a fun and easy brain training game.

Click each tile to reveal it's special icon and try and match it with it's matching pair.

The objective is to match all the tiles.

Test your memory by completing the game in as few moves as possible.

Test your reaction speed by completing the game in as little time as possible.

Your score is calculated by combining your elapsed time and number of moves

The lowest score wins!

How to play

  1. when you're ready to play, click the start button to start the timer.
  2. click the tile to reveal it's special icon
  3. click another tile to reveal it's icon and try to match each pair
  4. Each correct match will remain face up
  5. Each incorrect match will hide the tiles again
  6. A move is an attempt to match two tiles
  7. Continue to match tiles until all tiles have been successfully paired and remain revealed
  8. The game ends when you match all the pairs
  9. Your score will be revealed upon game completion
  10. Click the end game button to stop the game at any point
  11. Click the new game button to play again
